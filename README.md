# Netflix Content Analysis using Power BI

This project analyzes Netflix’s content catalog to understand how the platform distributes its movies and TV shows across genres, countries, and time.

---

## Problem Statement
With a rapidly growing content library, it becomes difficult to understand:
- What type of content Netflix focuses on
- How content is distributed across countries and genres
- Whether the platform is more movie-driven or series-driven
- How content additions have evolved over time

This dashboard aims to simplify these questions through visual analysis.

---

## Project Objectives
- Analyze the distribution of Movies vs TV Shows
- Identify top genres and content categories
- Understand country-wise content availability
- Analyze content release trends over time
- Provide a high-level overview of Netflix’s content strategy

---

## Dataset Overview
- Source: Public Netflix titles dataset
- Contains information on:
  - Title
  - Type (Movie / TV Show)
  - Genre
  - Country
  - Release year
  - Date added to Netflix
  - Duration and rating

The dataset was cleaned and prepared for visualization in Power BI.

---

## Key Analysis & Metrics
- Total titles on Netflix
- Movies vs TV Shows split
- Genre-wise content distribution
- Country-wise content availability
- Content addition trend over years

---

## Tools Used
- Power BI Desktop
- Data cleaning & transformation
- Interactive visuals and slicers
